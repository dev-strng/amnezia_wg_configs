# AmneziaWG Config Generator

Веб-приложение для автоматического сканирования Cloudflare WARP-эндпоинтов и генерации конфигов для AmneziaWG 2.0.

## Возможности

- **Сканер эндпоинтов**: Многопоточная проверка IP и портов Cloudflare WARP.
- **Генератор конфигов**: Создание `.conf` файлов с параметрами обфускации AmneziaWG.
- **WARP API**: Регистрация новых аккаунтов и получение ключей автоматически.
- **Интерактивный интерфейс**: Отслеживание прогресса сканирования в реальном времени.

## Требования

- Python 3.9+
- Работающий `ping` в системе (для ICMP проверки)

## Быстрый запуск

```bash
chmod +x run.sh
./run.sh
```

После запуска приложение будет доступно по адресу: `http://localhost:8000`

## Структура проекта

- `backend/`: FastAPI бэкенд (логика сканирования и генерации).
- `frontend/`: Одностраничное приложение (Vanilla JS).
- `run.sh`: Скрипт для настройки окружения и запуска.
